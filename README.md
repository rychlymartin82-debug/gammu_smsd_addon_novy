# Gammu SMSD Add-on for Home Assistant

This custom add-on allows Home Assistant to send SMS messages via a Huawei USB modem using Gammu.

## Features

- Sends SMS via shell script
- Uses Gammu inside Docker
- Accepts message text as parameter
- Designed for Home Assistant OS or Supervised

## Installation

1. Add this repository to Home Assistant Add-on Store 2. Install the add-on 3. Start the service 4. Use `shell_command` to send SMS
